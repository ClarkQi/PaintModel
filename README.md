# PaintModel

模仿涂涂乐，核心就是把屏幕四边形坐标投影到UV空间。基于Vuforia的AR SDK。

原理地址：http://www.arvrschool.com/read.php?tid=537&fid=23

P.S. 非商业项目，只为了提供一个大概思路，如何实现剩余的功能变成一个商业项目请不要问我……

使用方法：
打开Scenes中的场景，打开scene，把摄像头对准识别的图片，然后对识别图上色，上面的模型也会跟随着上色。
